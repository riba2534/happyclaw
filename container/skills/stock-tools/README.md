# 股票分析助手 Skills

这是一个专门为 HappyClaw 平台设计的股票分析助手 Skills 包。提供专业的股票分析功能，包括实时行情获取、历史数据查询、技术指标计算、新闻搜索与情绪分析、投资组合管理和价格提醒等功能。

## 功能特性

### 🏢 核心功能

| 工具名称 | 功能描述 |
|---------|---------|
| `get_stock_quote` | 获取股票实时行情数据 |
| `get_history_data` | 获取股票历史 K 线数据 |
| `calculate_indicator` | 计算技术指标（MA、EMA、MACD、RSI、BOLL、KDJ、VOL） |
| `search_news` | 搜索股票相关新闻 |
| `analyze_sentiment` | 情绪分析（利好/利空/中性） |
| `monitor_portfolio` | 监控投资组合 |
| `add_position` / `remove_position` | 管理持仓 |
| `set_alert` / `list_alerts` / `delete_alert` | 价格提醒管理 |

### 📊 技术指标

- **MA**：移动平均线（支持自定义周期）
- **EMA**：指数移动平均线
- **MACD**：平滑异同移动平均线
- **RSI**：相对强弱指标
- **BOLL**：布林带通道
- **KDJ**：随机指标
- **VOL**：成交量分析

### 💾 数据存储

- **行情数据缓存**：5分钟 TTL
- **历史数据缓存**：1小时 TTL
- **新闻缓存**：30分钟 TTL

## 安装与使用

### 项目结构

```
stock-tools/
├── src/
│   ├── index.ts         # 主入口文件
│   ├── quote.ts         # 实时行情获取
│   ├── history.ts       # 历史数据获取
│   ├── indicator.ts     # 技术指标计算
│   ├── news.ts          # 新闻搜索
│   ├── portfolio.ts     # 投资组合管理
│   └── alert.ts         # 价格提醒
├── dist/                # 编译后文件
├── tsconfig.json
├── package.json
└── README.md
```

### 编译

```bash
cd container/skills/stock-tools
npm install
npm run build
```

### 使用说明

在 HappyClaw 中，这些工具会自动通过符号链接发现。你可以直接在对话中使用：

```
帮我查一下贵州茅台 (600519.SS) 的实时行情
帮我分析一下 AAPL 的技术指标
帮我搜索关于特斯拉的新闻
```

## 数据来源

### 国内股票（A股）

- **东方财富网**：实时行情、新闻资讯
- **新浪财经**：财经新闻、公司公告
- **财联社**：实时电报、重大事件

### 国外股票（美股）

- **Yahoo Finance**：实时行情、历史数据
- **Bloomberg**：新闻资讯
- **Reuters**：市场分析

## 风险提示

- 本工具仅供投资分析参考，不构成投资建议
- 市场有风险，投资需谨慎
- 数据仅供参考，不保证完全准确
- 投资决策应由投资者独立做出

## 开发与维护

### 新增功能

1. 在 `src/` 目录下创建新的模块文件
2. 在 `src/index.ts` 中导出工具定义
3. 配置 `package.json` 的依赖
4. 运行 `npm run build` 编译

### 代码规范

- 使用 TypeScript
- 代码风格遵循 HappyClaw 项目规范
- 保持函数简洁和单一职责
- 添加适当的错误处理
